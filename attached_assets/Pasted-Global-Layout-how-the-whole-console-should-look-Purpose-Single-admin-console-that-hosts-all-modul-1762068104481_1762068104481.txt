Global Layout (how the whole console should look)
Purpose: Single admin console that hosts all modules. Clean, consistent, modular, accessible.
Core features
Persistent Sidebar (collapsible) with top logo (⚡ Electrocom) and module links: Dashboard, Documents, Clients, AMC, Tenders, Projects, Tasks, Employees & Contract Workers, Attendance, Payroll, Reports, Settings, Notifications.
Header: page title, breadcrumb, global search (universal across modules), notification bell (unread count), quick toggles (date-range picker), user avatar dropdown (profile, preferences, logout), light and dark theme toggler.
Main area: consistent grid-based content region. Each module uses cards, tables, forms, modals, slide-overs. Use responsive breakpoints.
Global utilities: theme (light/dark toggle), toast notifications, skeleton loaders, server-side pagination and filtering.

Visual / Interaction
Tailwind tokens: rounded-lg cards, soft shadows, neutral gray backgrounds, primary accent sky-500. Use Lucid icons.
Sidebar collapses to icons on mobile; mobile uses bottom navigation for main modules.
All tables support sortable columns, column toggles (show/hide).
Provide “quick create” sticky CTA for adding clients, AMCs, tenders, tasks.
